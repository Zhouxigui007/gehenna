# gehenna

Awesome ES6 polyfills and mega helpers for for Adobe scripting

## Installation

```bash
npm i gehenna
```

```js
// To automatically load all utilies:
async require('gehenna').default.init()
```

### JSON

- `JSON.stringify()`
- `JSON.parse()`

### Arrays

- `Array.find()`
- `Array.map()`
- `Array.includes()`
- `Array.forEach()`
- `Array.flat()`
- `Array.filter()`

### Objects

- `Object.keys()`
- `Object.assign()`
- `Object.entries()`

### Strings

- `String.trim()`
- `String.padLeft()`
- `String.padRight()`

### Console

- `console.log()`
- `console.error()`
- `console.debug()`
